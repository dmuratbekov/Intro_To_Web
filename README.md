# INTRO TO WEB PROGRAMMING

This course provided a comprehensive introduction to full-stack web development. Throughout the semester, I progressively developed skills in programming fundamentals, front-end development, back-end development, database management, and application integration.

The course began with core programming concepts in Python and gradually advanced toward building interactive web applications using HTML, CSS, JavaScript, Flask, and PostgreSQL.

---

## Programming Foundations (Python)

The first stage of the course focused on Python fundamentals. This part established a strong foundation in logical thinking, problem solving, and structured programming.

Topics covered:
- Installing and configuring Python
- Working with virtual environments
- Data types and operations
- Control flow (conditions and loops)
- Functions and modular programming
- Error handling and debugging
- Working with modules and libraries
- Object-Oriented Programming (OOP)
- Data structures (lists, dictionaries, tuples, sets)
- File handling
- Working with dates and times
- Regular expressions
- JSON processing

Skills gained:
- Writing clean and modular code
- Structuring applications using functions and classes
- Processing files and structured data
- Handling exceptions properly
- Organizing projects efficiently

---

## Introduction to Web Development & Flask

After building a programming foundation, the course transitioned into web development concepts and backend frameworks.

Topics covered:
- How the web works (client-server architecture)
- HTTP basics
- Creating Flask applications
- Routing and request handling
- Rendering templates
- Running local development servers

Skills gained:
- Building backend web applications
- Understanding request-response cycles
- Connecting Python logic with HTML templates
- Managing development environments

## Database Fundamentals (PostgreSQL)

The course concluded with relational database management and SQL.

Topics covered:
- Introduction to databases
- Creating and managing tables
- Data manipulation (INSERT, UPDATE, DELETE)

- Filtering with WHERE
