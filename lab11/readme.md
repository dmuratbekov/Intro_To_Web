# Data Structures in Python
This laboratory work is dedicated to studying Python built-in data structures and their practical usage. Data structures are a core concept in programming, as they allow efficient storage, organization, and manipulation of data.

In this lab, I explored the most commonly used Python data structures: lists, dictionaries, sets, and tuples, along with their methods and operations.

## Objectives
The main goals of this laboratory work were:
- To understand why data structures are important in programming
- To learn where and how different data structures are used in real-world applications
- To practice working with built-in Python data structures
- To apply common methods and operations for data manipulation
- To solve practical exercises using lists, dictionaries, sets, and tuples

## Practical Exercises
The laboratory work included several practical exercises:
- Working with lists (basic operations, slicing, indexing)
- Working with dictionaries (adding, updating, and removing data)
- Performing set operations (union, intersection, difference)
- Using tuple methods
- Solving a challenge task with nested data structures (lists inside dictionaries)

Each task was implemented step by step, and correct outputs were verified.
